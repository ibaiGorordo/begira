# begira examples

## hello_begira

Starts the begira viewer with one Python script.

```bash
cd /Users/ibaigorordo/Desktop/myprojects/begira
python -m pip install -e .
python examples/hello_begira.py
```

If you haven't built the frontend yet:

```bash
cd /Users/ibaigorordo/Desktop/myprojects/begira/frontend
npm install
npm run build
```
